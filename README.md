# Groq Chat Desktop

Μια σύγχρονη desktop εφαρμογή συνομιλίας με AI, χτισμένη με **Python** και **PySide6**, σχεδιασμένη για άμεση επικοινωνία με το **Groq API**.

Η εφαρμογή φτιάχτηκε με στόχο να προσφέρει ένα καθαρό, όμορφο και πρακτικό περιβάλλον εργασίας για συνομιλία με AI μοντέλα μέσω cloud.  
Δίνει έμφαση στην ευχρηστία, στην υποστήριξη ελληνικής γλώσσας, στη σωστή εμφάνιση κώδικα μέσα στις απαντήσεις και σε μια πιο επαγγελματική desktop εμπειρία.

---

## Χαρακτηριστικά

- Άμεση σύνδεση με το **Groq API**
- Χωρίς RAG pipeline
- Χωρίς Ollama ή άλλο τοπικό inference server
- Ανανέωση και επιλογή διαθέσιμων μοντέλων από το UI
- Μεγάλο ενιαίο πλαίσιο συνομιλίας
- Υποστήριξη **ελληνικών απαντήσεων**
- Βελτιωμένη εμφάνιση κώδικα μέσα στις απαντήσεις
- Χρήση γραμματοσειράς **Consolas** στα code blocks
- Επιλεκτική αντιγραφή μόνο των code blocks
- Light / Dark themes
- Light theme ως προεπιλογή
- Ενσωματωμένα logs για διάγνωση σφαλμάτων
- Εξαγωγή συνομιλίας
- Παραμετροποίηση generation settings
